To setup pants in your repo: 
      curl https://pantsbuild.github.io/setup/pants.sh > pants
      chmod +x pants
      touch pants.ini
The first time you run the new ./pants script it will install the latest pants version and then run it. On subsequent runs it will find the already installed pants and run it. To pin the pants version, modify the PANTS_REQUIREMENT variable in the pants script like so: 
      PANTS_REQUIREMENT="pantsbuild.pants==0.0.34"
